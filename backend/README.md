# 后端服务
使用[kaxiluo/fastapi-skeleton](https://github.com/kaxiluo/fastapi-skeleton)

## 安装
### 环境准备
开发环境为Python 3.11，如果使用其他环境运行，可能会出现依赖版本不兼容的问题，请自行调试解决

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置env
在`.env.example`中配置好相关的参数
配置好之后修改文件名
```bash
cp .env.example .env
```

### 运行
```bash
python main.py
```
